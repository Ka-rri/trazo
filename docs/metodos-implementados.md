# Métodos Numéricos a Implementar - Proyecto Trazo

Este documento detalla la lista de algoritmos numéricos que se integrarán en la librería **Trazo**.

| Nombre del Método | Categoría | Descripción Breve | Estado |
|---|---|---|---|
| **Bisección** | Raíces de Ecuaciones | Encuentra raíces dividiendo el intervalo a la mitad. | Pendiente |
| **Newton-Raphson** | Raíces de Ecuaciones | Método iterativo basado en la derivada de la función. | Pendiente |
| **Secante** | Raíces de Ecuaciones | Variante de Newton que no requiere la derivada explícita. | Pendiente |
| **Trapecio** | Integración Numérica | Aproxima el área bajo la curva usando trapecios. | Pendiente |
| **Simpson 1/3** | Integración Numérica | Utiliza parábolas para aproximar integrales con mayor precisión. | Pendiente |
| **Lagrange** | Interpolación | Construye un polinomio que pasa por puntos específicos. | Pendiente |
