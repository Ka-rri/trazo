# Glosario

| Término | Definición |
|---|---|
| Algoritmo | Secuencia de pasos lógicos y finitos para resolver un cálculo o problema ya sea |
| Aproximación | Valor cercano al resultado matemático exacto, usado cuando el calculo perfecto es muy complejo |
| Bisección | Método para encontrar raíces que divide un intervalo a la mitad repetidamente |
| Convergencia | Cuando las respuestas de un algoritmo se acercan cada vez más a la solución correcta |
| Cota de error | El límite máximo que nos dice qué tan equivocada podría estar nuestra aproximación |
| Coeficiente de determinación (R²) | Medida que indica qué tan bien un modelo de regresión explica los datos observados |
| Cuadratura de Gauss | Método numérico para aproximar integrales utilizando puntos y pesos seleccionados estratégicamente |
| Derivada | Operación matemática que calcula la tasa de cambio de una función en un punto específico |
| Divergencia | Cuando un algoritmo falla y sus resultados se alejan cada vez más de la respuesta real |
| Descomposición LU | Técnica que factoriza una matriz en una matriz triangular inferior (L) y una superior (U) |
| Diferencia centrada | Aproximación de una derivada utilizando valores a ambos lados del punto de interés |
| Diferencias divididas | Procedimiento usado en la interpolación de Newton para calcular coeficientes del polinomio interpolante |
| Error absoluto | La diferencia directa (resta) entre el valor matemático real y nuestro valor aproximado |
| Error relativo | El error absoluto dividido por el valor real. Sirve para ver el error en forma de porcentaje |
| Error relativo aproximado | Estimación del error calculada a partir de dos iteraciones consecutivas de un método numérico |
| Extrapolación | Técnica para calcular o estimar un valor que está fuera del rango de nuestros datos conocidos |
| Interpolación | Método para adivinar un valor desconocido que está en medio de dos datos que sí conocemos |
| Iteración | Cada "vuelta" o ciclo que da un algoritmo hasta acercarse a la solución deseada |
| Matriz | Arreglo de números en filas y columnas, muy útil para resolver sistemas de ecuaciones |
| Método de Runge-Kutta (RK4) | Método numérico de cuarto orden para aproximar soluciones de ecuaciones diferenciales |
| Orden del método | Medida de la rapidez con la que disminuye el error al reducir el tamaño del paso |
| Pivoteo | Técnica usada en matrices para reordenar las filas y evitar errores graves de cálculo |
| Precisión | La cantidad de decimales correctos o el nivel de exactitud que tiene nuestro cálculo |
| Raiz | El valor exacto donde una función matemática cruza la línea del cero (es decir, donde f(x) = 0) |
| Redondeo | Reducir la cantidad de decimales de un número aproximándolo al valor más cercano |
| Regresión | Técnica estadística utilizada para modelar la relación entre variables |
| Sistema lineal | Conjunto de ecuaciones que se resuelven juntas, muy común en métodos numéricos |
| Spline cúbico | Método de interpolación que utiliza polinomios cúbicos por tramos para generar curvas suaves |
| Tolerancia | El nivel de error máximo que estamos dispuestos a aceptar para que el programa se detenga |
